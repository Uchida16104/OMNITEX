
# Unified Laravel + Vue + Multi‑Runtime Blueprint

A polyglot full-stack template integrating Laravel, Vue.js, HTMX, Tailwind CSS,
Alpine.js, Rust, Python, ASP.NET, Zig, Mojo, Carbon, F*, Dafny, Processing,
Mermaid, and LaTeX.

## Architecture

- Frontend: Vercel (`vercel.json`)
- Backend: Render (`render.yaml`)
- Centralized orchestration through Dockerfile and YAML configuration
